# Usage Guide

This document explains how to use the project.

## 1. Clone the Repository
```bash
git clone https://github.com/your-username/cncf-krm.git
cd cncf-krm
```

## 2. Install dependencies
```bash
pip install -r requirements.txt
```

## 3. Run the application
```bash
python main.py
```
